USDX Token API Reference

API endpoints for interacting with USDX token smart contract on Sepolia testnet.

Read Endpoints

Endpoint Method Purpose
/api/token GET Get token name, symbol, supply, shares, multiplier, and pause status
/api/account/:address GET Get address balance, shares, and blocklist status
/api/has-role/:account/:role GET Check if account has specific role
/api/transaction/:txHash GET Get transaction details and status

Transfer Endpoints

Endpoint Method Purpose
/api/transfer POST Transfer tokens to address
/api/mint POST Mint new tokens to address (requires MINTER_ROLE)
/api/burn POST Burn tokens from address (requires BURNER_ROLE)

Admin Endpoints

Endpoint Method Purpose
/api/pause POST Pause all token transfers (requires PAUSE_ROLE)
/api/unpause POST Unpause token transfers (requires PAUSE_ROLE)
/api/block POST Block account from transfers (requires BLOCKLIST_ROLE)
/api/unblock POST Unblock account (requires BLOCKLIST_ROLE)
/api/reward-multiplier POST Update reward multiplier (requires ORACLE_ROLE)
/api/grant-role POST Grant role to account (requires DEFAULT_ADMIN_ROLE)
/api/revoke-role POST Revoke role from account (requires DEFAULT_ADMIN_ROLE)
