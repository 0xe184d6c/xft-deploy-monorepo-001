Ethereum ABI to OpenAPI Converter

Convert your Ethereum smart contract ABI to an OpenAPI specification

How to Use

This service provides an API endpoint that converts Ethereum smart contract ABIs to OpenAPI specifications.

Send a POST request to /generateSpec with your ABI JSON in the request body.

Example curl request:

curl -X POST http://localhost:8000/generateSpec \
    -H "Content-Type: application/json" \
    -d @your-abi.json
